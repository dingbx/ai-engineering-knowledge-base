# OpenClaw Creator Joins OpenAI

**Date:** 2026-02-14 (announced)
**Category:** Business

---

## Summary

Peter Steinberger, creator of the viral OpenClaw AI agent (234K+ GitHub stars), announced he is joining OpenAI. The OpenClaw project will be moved to an open-source foundation.

---

## Background

OpenClaw (originally Clawdbot, then Moltbot) grew faster than Docker, Kubernetes, or React:
- 9K stars in first 24 hours
- 214K+ stars by February 2026
- 234K+ stars by March 2026

---

## Implications

1. **Validation**: OpenAI sees value in autonomous agent paradigm
2. **Foundation**: Project continues under community governance
3. **Talent**: Key agent developer now at leading AI lab
4. **Direction**: Signals OpenAI interest in agent capabilities
