# Claude Code

> Terminal-first AI coding assistant for large-scale refactoring

**Context:** 200K tokens
**Price:** ~$20/month (Claude Pro)
**Best For:** 50K+ LOC codebases

---

## Overview

Claude Code excels at terminal-based, multi-step refactors. It can navigate and understand entire repositories, making coherent multi-file changes.

---

## Key Features

### Massive Context Window
- 200K tokens (vs 32K-128K for competitors)
- Entire codebases fit in context
- No need to manually select files

### Repository Understanding
- Navigates full project structure
- Understands relationships between files
- Coherent cross-file changes

### Terminal-First
- CLI-based workflow
- Great for server/remote work
- Integrates with existing terminal tools
