# Cursor

> AI-native IDE built for developers working with large codebases

**Type:** Standalone IDE (VS Code fork)
**Website:** [cursor.sh](https://cursor.sh)

---

## Overview

Unlike tools that bolt AI onto existing editors, Cursor is built from the ground up as an AI-native IDE. It's a fork of VS Code with deeply integrated AI capabilities.

---

## Key Features

### Project-Wide Context
- Understands entire codebase
- Multi-file awareness
- Better than tools limited to open files

### Composer Mode
- Rapid prototyping
- Complex multi-file edits
- Strong for greenfield development

### Model Selection
- Choose between different AI models
- Access to Claude, GPT, and others
- Switch based on task needs
