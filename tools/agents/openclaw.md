# OpenClaw

> Viral open-source AI agent with unprecedented growth and significant security concerns

**GitHub:** 234K+ stars (fastest growth ever)
**Status:** Active, creator joined OpenAI
**Security:** 512 vulnerabilities identified

---

## Overview

OpenClaw is an autonomous AI agent that executes real-world tasks via messaging platforms. Unlike chatbots that just respond to queries, OpenClaw acts.

## What It Does

Runs locally and integrates with LLMs (Claude, DeepSeek, GPT) to:
- Read emails
- Manage calendars
- Run terminal commands
- Deploy code
- Maintain memory across sessions

### Interface
Accessed via chatbot in messaging services:
- Signal
- Telegram
- Discord
- WhatsApp

## Security Concerns

### Audit Results (Jan 2026)
- **512 vulnerabilities** identified
- **8 critical** severity
- Cisco called it "a security nightmare"

### Maintainer Warning
> "If you can't understand how to run a command line, this is far too dangerous of a project for you to use safely."

### Risks
- Arbitrary code execution
- Access to local system
- Messaging platform integration
- Autonomous decision-making

## Why It Matters

1. **Growth**: Faster than Docker, Kubernetes, or React

## Should You Use It?

**Consider if:**
- You understand the security risks
- You can audit the code
- You have isolated environment
- You need autonomous task execution

**Avoid if:**
- You're not security-savvy
- You'd run it on production systems
- You can't monitor its actions
